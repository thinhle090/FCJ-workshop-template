# Serverless Job Application Tracker

#### Overview

This workshop walks you through building a **fully serverless** web application on AWS: a job application tracker with user authentication, secure CV storage, automatic email reminders, operational monitoring and complete audit logging.

By the end you will know how to combine Cognito, API Gateway, Lambda, DynamoDB, S3, KMS, Amplify, WAF, Route 53, EventBridge, SES, SQS, CloudWatch and CloudTrail into a working system — for under $1 in actual cost across the entire development process.
