### Week 4 Achievements:
* Simplified Computing with Amazon Lightsail:
  * Create pre-configured template structures (Blueprints) such as WordPress, LAMP, and MEAN stacks with fixed-cost, easily predictable models.
  * Manage infrastructure backups (Snapshots), firewall rules, and static IP allocation directly from the Lightsail console.

* Container Deployment with Lightsail Containers:
  * Package application source code into Docker Images and push it to Lightsail Container Services.
  * Configure multi-node deployments, manage container scaling, and monitor container health in real time.

* Flexible Scaling with EC2 Auto Scaling:
  * Build Launch Templates and Auto Scaling Groups (ASGs) to automate the scaling of computing resources.
  * Define Target Tracking policies triggered by CPU usage, network traffic, or custom schedules.
  * Integrate ASG with Application Load Balancers (ALB) for smooth traffic routing based on target health checks.

* Content delivery network with Amazon CloudFront:
  * Create CloudFront Distributions sourced from S3 and ALB to accelerate global resource delivery.
  * Configure caching policies, set time-to-live (TTL) values, SSL/TLS security certificates, and geo-restricted routing.
