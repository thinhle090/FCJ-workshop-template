### Week 1 Objectives:

* Become familiar with the internship environment, working process, and AWS learning roadmap.
* Understand the fundamentals of Cloud Computing and AWS.
* Learn about AWS Global Infrastructure, including Regions, Availability Zones, and Edge Locations.
* Explore the major AWS service categories and their purposes.
* Become familiar with the AWS Management Console and official AWS learning resources.

### Week 1 Achievements:

* Understood the fundamental concepts of Cloud Computing and the cloud service models (IaaS, PaaS, and SaaS).
* Learned about AWS Global Infrastructure, including Regions, Availability Zones, and Edge Locations.
* Gained an overview of the major AWS service categories, including Compute, Storage, Database, Networking, Security & Identity, and Management & Monitoring.
* Understood the functions and common use cases of AWS services such as Amazon EC2, Amazon S3, Amazon IAM, Amazon VPC, Amazon RDS, Amazon Route 53, Amazon CloudWatch, and AWS Lambda.
* Became familiar with the AWS Management Console and learned how to navigate AWS services efficiently.
* Built a solid foundation for hands-on AWS labs and cloud deployment activities in the following weeks.
