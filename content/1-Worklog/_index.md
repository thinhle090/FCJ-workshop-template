This page presents the worklog throughout my internship at **Amazon Web Services (AWS) Vietnam**. The internship program was completed over **12 weeks**, from **20/04/2026** to **05/07/2026**. During this period, I progressively learned and practiced AWS core services such as **Amazon VPC, Amazon EC2, Amazon S3, Amazon RDS, AWS IAM, Amazon CloudFront, Amazon Route 53**, and various cloud security solutions. After completing the training phase, I participated in the implementation of an **AWS-based E-commerce Website** project, which included requirement analysis, system architecture design, infrastructure deployment, application implementation, database configuration, security enhancement, system testing, and the completion of technical documentation, reports, and project demonstrations. The detailed weekly activities are presented below.

Typically, and as a standard practice, an internship worklog is completed over approximately **three months**. The weekly activities during my internship are summarized as follows:

**Week 1:** [Registered an AWS account and became familiar with fundamental AWS concepts and introductory hands-on labs.](1.1-week1/)

**Week 2:** [Studied and practiced networking services, Amazon EC2, IAM Roles, and Amazon CloudWatch.](1.2-week2/)

**Week 3:** [Learned about AWS storage services, databases, and in-memory caching solutions.](1.3-week3/)

**Week 4:** [Explored application deployment using Amazon Lightsail, EC2 Auto Scaling, and Amazon CloudFront.](1.4-week4/)

**Week 5:** [Studied Amazon Route 53, AWS Networking Workshop, and Edge Computing with CloudFront and Lambda@Edge.](1.5-week5/)

**Week 6:** [Learned about Windows workloads on AWS, AWS Managed Microsoft AD, and highly available web application architectures.](1.6-week6/)

**Week 7:** [Initiated the project by analyzing requirements and designing the architecture for an AWS-based e-commerce website.](1.7-week7/)

**Week 8:** [Set up the AWS infrastructure, including VPC, subnets, Amazon EC2, and secure network connectivity.](1.8-week8/)

**Week 9:** [Deployed the web application to Amazon EC2, configured the runtime environment, and performed functional testing.](1.9-week9/)

**Week 10:** [Built the database using Amazon RDS and configured Amazon S3 for product image storage.](1.10-week10/)

**Week 11:** [Enhanced system security by configuring IAM, HTTPS, Security Groups, and conducting basic security assessments.](1.11-week11/)

**Week 12:** [Performed system testing, resolved remaining issues, finalized technical documentation, and prepared the project presentation and demonstration.](1.12-week12/)
