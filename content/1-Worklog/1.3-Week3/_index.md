### Week 3 Achievements:

* Object-Oriented Hosting with Amazon S3:
  * Create and manage S3 Buckets, setting up file lifecycle rules to optimize hosting costs.
  * Configure Static Website Hosting and successfully integrate it with custom domain setups.
  * Control data security by precisely defining Bucket Policies and Access Control Lists (ACLs).

* Structured Database Services with Amazon RDS:
  * Deploy popular administrative database tools (MySQL, PostgreSQL) and configure Multi-AZ for increased availability.
  * Set up automatic backup windows, maintenance schedules, and create read-only replicas to reduce query traffic.
  * Isolate database entities within private subnets and restrict network access using targeted security groups.

  * NoSQL Database Services with Amazon DynamoDB:
* Design optimized database tables using Partition Keys and Sort Keys to streamline data querying.
  * Evaluate between Provisioned and On-demand capacity modes to balance performance and cost.
  * Use DynamoDB Streams to monitor, log, and respond to data changes in real time.

* Internal caching with Amazon ElastiCache:
  * Deploy Redis and Memcached clusters to accelerate access for heavy application queries.
  * Analyze core caching strategies such as Lazy Loading and Write-through to maintain strict data consistency.
